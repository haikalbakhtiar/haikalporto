# Haikal — Portfolio v2 (Multi-Page)

Built with Next.js 15 + TypeScript + Tailwind CSS + Framer Motion.

## Pages
- `/` — Home (Hero, About, Skills, Work Cards, Travel, Contact)
- `/data-analytics` — Full 6-step analytics case study with datasets
- `/system-testing` — Full 6-phase testing case study with datasets
- `/service-level` — Full 5-phase SLM case study with datasets

## Get Started
```bash
npm install
npm run dev
```
Open http://localhost:3000

## Deploy
Push to GitHub → connect at vercel.com (zero config for Next.js)
